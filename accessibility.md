# Accessibility
This statement was created on 11 May 2026 using the [W3C Accessibility Statement Generator Tool](https://www.w3.org/WAI/planning/statements/).

## Conformance status
The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. 

A11y Lab Resources is fully conformant with WCAG 2.2 level AA. Fully conformant means that the content fully conforms to the accessibility standard without any exceptions.

## Additional accessibility considerations
We link to external accessibility resources and tools that are maintained by other organisations. While we try to share resources that are as accessible as possible, we cannot guarantee the accessibility of external websites or content
